### Krill Lambdas Cookbook

These are python scripts that can be used as [lambdas in Krill](https://krillswarm.com/posts/2026/01/25/lambda-executor/). They are meant to be examples and can be modified to fit your needs. They are not meant to be used as-is in production.
Feel free to open a pull request to add your own lambdas to the cookbook!

Krill can execute a lambdas based on a trigger and then return the result to be used in your swarm. This allows you to run custom code and integrate with other systems and APIs.
